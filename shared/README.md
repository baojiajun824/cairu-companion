# cairu-common

Shared utilities for cAIru Base Station services.

## Installation

```bash
pip install -e .
```

## Modules

- `config` - Settings management using pydantic-settings
- `logging` - Structured logging with structlog
- `metrics` - Prometheus metrics
- `models` - Shared Pydantic models for inter-service communication
- `redis_client` - Redis Streams client wrapper
